Introduction

What the article/code snippet does, why it's useful, the problem it solves etc.

Visual Studio 2010 is an excellent tool, but default project template and wizards continue to encourage bad design practices. This article shows a better structure for your projects that will reduce the coupling between the different layers.

To give the article a bit more realism, the text comes along with set of projects to illustrate the point. The projects demonstrate how to build a REST based service atop of Windows Communication Foundation (WCF), how to host the service in two types of hosting and a sample web client application.
